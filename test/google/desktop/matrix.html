      Description

      The Matrix is an American media franchise consisting of four feature films, beginning with The Matrix and continuing with three sequels, The Matrix Reloaded, The Matrix Revolutions, and The Matrix Resurrections. The first three films were written and directed by The Wachowskis and produced by Joel Silver. Wikipedia
